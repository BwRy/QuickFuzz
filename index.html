QuickFuzz

An experimental grammar fuzzer in Haskell using QuickCheck

View project on GitHub

QuickFuzz is grammar fuzzer powered by QuickCheck, Template Haskell and specific libraries from Hackage to generate many complex file-formats like Jpeg, Png, Svg, Xml, Zip, Tar and many more!. QuickFuzz is open-source (GPL3) and you can find it here

Bugs (lost and) found

  • Firefox (failed assert in gif loader, undisclosed)
  • Gdk-Pixbuf (CVE-2015-4491, CVE-2015-7674, CVE-2015-7673, undisclosed)
  • Jasper (CVE-2015-5203)
  • Unzip (multiple memory issues)
  • GNU Tar (out-of-bound access)
  • Optipng (buffer overflow, use-after-free)
  • Libtiff (CVE-2015-7313)
  • Libarchive (big allocation in tar handling)

Quick introduction to QuickFuzz

Let's generate, fuzz and test for crashes giftopnm 10 times with a maximum structural size of 300:

$ QuickFuzz Gif c.gif "/usr/bin/giftopnm c.gif" -a zzuf -t 10 -s 300
zzuf[s=1806,r=0.004:1e-06]: signal 11 (SIGSEGV)
zzuf[s=1807,r=0.004:1e-06]: signal 11 (SIGSEGV)
+++ OK, passed 10 tests.

In this case, two crashes were found (actually, they are produced by the same bug).
