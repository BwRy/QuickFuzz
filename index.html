QuickFuzz

An experimental grammar fuzzer in Haskell using QuickCheck

View project on GitHub

QuickFuzz is a grammar fuzzer powered by QuickCheck, Template Haskell and specific libraries from Hackage to generate many complex file-formats like Jpeg, Png, Svg, Xml, Zip, Tar and more!. QuickFuzz is open-source (GPL3) and it can use other bug detection tools like zzuf, radamsa, honggfuzz and valgrind.

Bugs lost and found

  • Firefox (failed assert in gif loader, CVE-2015-7194, CVE-2015-7216, CVE-2015-7217)
  • Gdk-Pixbuf (CVE-2015-4491, CVE-2015-7674, CVE-2015-7673, undisclosed)
  • Mplayer (several reads out-of-bound in mplayer 1.1)
  • Jasper (CVE-2015-5203)
  • Unzip (CVE-2015-7696, CVE-2015-7697)
  • GNU Tar (out-of-bound access)
  • Optipng (CVE-2015-7802, CVE-2015-7801)
  • Libtiff (CVE-2015-7313)
  • Libarchive (big allocation in tar handling)

Quick introduction to QuickFuzz

Let's generate, fuzz and test for crashes giftopnm using zzuf:

$ QuickFuzz Gif "/usr/bin/giftopnm @@" -a zzuf -t 100 -s 100
 zzuf[s=2687,r=0.004:1e-06]: signal 11 (SIGSEGV)
 +++ OK, passed 100 tests.
